## 我们在做什么

Yuxino Labs 用来集中管理我们的产品仓库、后端服务、内部工具和发布基础设施。

我们偏爱小团队、快速反馈、可维护的基础设施，以及真正顺手的内部工具。

- 内部工具不是摆设，要做到团队愿意天天用
